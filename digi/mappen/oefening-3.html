Oefeningen

We gaan een backup maken van bepaalde bestanden. Die backup kunnen we bewaren op een usb-stick.
Een backup op usb gaan we ook weer terugzetten op de computer.
We zullen in de oefeningen enkele aspecten van 'opruimen' aan de orde laten komen.
  1. Maak een backup van de map 'Oefenmappen' via de Settings van Windows 10.

  2. Ga naar de usb-stick en maak een map 'Backup'.
    Maak handmatig een backup van de mappen 'Map-A', 'Map-B' en 'Map-C' op de usb-stick in de map 'Backup'.

  3. Maak een backup van de map 'Oefenmappen' op usb via de Settings van Windows 10.

  4. Verwijder de mappen 'Map-A', 'Map-B' en 'Map-C' op de computer.
    Zet de backup terug vanaf de usb-stick.
