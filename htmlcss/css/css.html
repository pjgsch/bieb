CSS

CSS staat voor Cascading Style Sheets, een programeertaal die zorgt voor de opmaak ('style') van HTML bestanden.
Als een pagina geen CSS voor de opmaak gebruikt, kan het resultaat er anders uitzien in verschillende browsers. De browser interpreteert dan de html-codes op een default manier. Dat wordt aangeduid als 'user agent stylesheet'.

Er zijn drie vormen, die we afzonderlijk zullen bekijken:

  • in-line styles : de CSS definities staan in de HTML-tag; we behandelen ze hier

  • interne styles : de CSS definities staan apart in de HTML-pagina; we behandelen ze hier

  • externe styles : de CSS definities staan in aparte files; we behandelen ze hier
